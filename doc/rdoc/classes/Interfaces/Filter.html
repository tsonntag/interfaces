Class Interfaces::Filter
In: lib/interfaces/filter.rb
Parent: Base

Base class for filters.

To implement collect path behaviour subclasses must overwrite do_filter_files(pathes) to implement collect path behaviour.

To implement single path behaviour subclasses must either implement do_filter_file(path) and target_path(path) or overwrite filter_file if the target_path cannot be derived from path (e.g. unzip)
