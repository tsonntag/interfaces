Class Interfaces::Builder
In: lib/interfaces/builder.rb
Parent: Object

Builds interfaces from given configuration config Config is a hash with the structure: config[interface_name][base_name][key] where base is a source, sink or filter Default config is Config::Interfaces.default

Usage:

interfaces = Interfaces::Builder.build do |b| 

  b.foo :bar, :from, :step1, :step2, :to

end

generates an interface named foo with source FromSource, filters Step1Filter and Step2Filter and sink ToSink using the configuration :bar

Example:

config has structure (here in yaml format) gd: 

  dir_source:
     dir: /home/hermes/data/sending/gd
     regexp: \AGD.*\.xml\Z
  pgp_encrypt_filter:
     pgp_encrypt_cmd: pgp ....
     public_keys: gd@t-mobile.de
     ...
  ftp_sink:
    host: gd
    password: bla
    remote_dir: file_from_hermes
  ftp_source:
    host: gd
    password: bla
    remote_dir: file_to_hermes

arvato_: 

  dir_source:
     dir: /home/hermes/data/sending/gd
     regexp: .*\.txt\Z
  mail_sink:
     recipients: mail@arvato.de
     subject: 'Files: %files'
     body: 'Here are yours files:\n %files'

# build interfaces interfaces = Interfaces::Builder.build do |b| 

  b.gd_sender   :gd, :dir, :pgp, :ftp
  b.gd_receiver :gd, :ftp, :pgp, :ftp
  b.arvato_sender :dir, :zip, :mail, :collect => true

end

# call interface gd_sender interfaces[:gd_sender].call
